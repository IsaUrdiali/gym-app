### ⚙ Para utilizar o projeto

1. Execute o clone no seu terminal.

```bash
git clone https://github.com/IsaUrdiali/gym-app.git
```

2. Instale as dependências do projeto 

```bash
npm install
```

3. Execute o comando de execução do projeto

```bash
npm run dev
```
